Leanring for EF Core 9.0

##Running Migrations
# Add a migration
dotnet ef migrations add InitialCreate --project OM.MFPTracker.Data --startup-project ConsoleApp.Test
dotnet ef migrations add InitBlazor --project OM.MFPTracker.Data --startup-project OM.MFPTracker.Web

# Update the database
dotnet ef database update --project OM.MFPTracker.Data --startup-project ConsoleApp.Test
dotnet ef database update --project OM.MFPTracker.Data --startup-project OM.MFPTracker.Web

##Delete the Database
dotnet ef database drop --project OM.MFPTracker.Data --startup-project ConsoleApp.Test

##Remove the Last Migration
dotnet ef migrations remove --project OM.MFPTracker.Data --startup-project ConsoleApp.Test


##delete the migrations folder & re-initialize
dotnet ef migrations add InitialCreate --project OM.MFPTracker.Data --startup-project ConsoleApp.Test
dotnet ef database update --project OM.MFPTracker.Data --startup-project ConsoleApp.Test
